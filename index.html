HTML

  • HTML stands for Hypertext Markup Language
  • HTML is made up of elements which work together to provide content and the basic framework for all pages on the web
  • The following 4 element types are commonly used to add content to pages:

Elements

  1. Text Elements
  2. Image Elements
  3. Header Elements
  4. List Elements
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • External CSS style sheet allows developers to keep all their CSS rules in a seperate file making design changes easier.

NOTE: We're using external CSS style sheet and it's prefered in most cases

  • When using an external style sheet, we link the file using a "link" element placed in the HTML's "head"
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The period preceding card in ".card{ }" designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • Assigning colors in CSS can be done in many ways, including by semantic references such as gray or blue. We can also refer to a color by its hexadecimal code, or hex #.
